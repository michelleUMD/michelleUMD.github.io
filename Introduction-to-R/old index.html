Introduction to R/RStudio

Author: Michelle

Hi! My name is Michelle, and I am a college student studying computer science and biology. Learning basic programming can be a useful skill for students in the life sciences, given the integration of technology and big data as part of the scientific field, and one useful programming language is R. This course will teach life science majors the basics of R programming language and how to use R with scientific datasets. We'll learn about functions, reading files, data cleaning, modeling, statistical tests, plotting, and R packages/libraries. No prior programming experience is assumed.

With every "lecture," there is both a slide deck that explains the programming concepts and the corresponding R code that shows the application of those concepts. This course was originally created by me to teach at my university, so I also include the homework I assigned my students in case you would like extra practice. 

Thanks for checking out my website, and I hope you enjoy this course! Good luck!
